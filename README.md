## **📝 필사 용지 메이커 (Manuscript Paper Maker)**

사용자가 입력한 텍스트를 분석하여 브라우저 환경에서 실측 A4 규격에 맞춘 **원고지 및 줄노트**로 자동 조판해 주는 단일 페이지 웹 애플리케이션(SPA)입니다.

단순히 화면을 렌더링하는 것을 넘어, 대용량 텍스트 입력 시에도 버벅임이 없도록 **바닐라 JS 기반의 메모리 재사용(DOM Recycling) 및 32비트 롤링 해시(Rolling Hash) 점진 업데이트** 기술을 채택하고 있으며, 외부 악성 마크업 유입을 100% 차단하는 **시큐어 코딩 규칙**이 적용되어 있습니다.

## **🚀 핵심 기능 (Key Features) **

1. **다단 조판 엔진**  
   * **원고지 모드:** 표준 20칸(가로 200자 원고지/ 세로 500자 표준형)을 비롯하여 10칸, 12칸, 14칸, 16칸, 18칸, 22칸, 24칸까지 용지 크기에 맞게 자동으로 행과 열의 격자를 연산하여 생성합니다.  
   * **줄 노트 모드:** 원고지 격자 없이 가로줄만 표시하여 보다 자유롭고 긴 호흡의 필사를 지원합니다.  
2. **원문 분석 및 한글 문장부호 압축 규칙 (Typography rules)**  
   * 행 끝에 도달했을 때 온점(.), 반점(,), 큰따옴표(”), 물음표(?), 느낌표(\!) 등의 문장부호가 위치할 경우, 다음 행 첫 칸으로 넘어가지 않고 현재 행 마지막 칸에 물리적으로 압축 합산(isSqueezed, squeezedPunct)하여 가독성을 높입니다.  
   * 구어체 대화 등에서 자주 쓰이는 연속 문장부호 쌍(.”, ?” 등)을 한 칸에 부드럽게 병합 조판합니다.  
3. **사용자 맞춤형 정밀 옵션**  
   * **따라 쓰기 지면 제공:** 연한 명조체 텍스트 가이드라인이 비쳐 보여 바른 글씨 연습을 도와줍니다.  
   * **빈 원고지 지면 제공:** 가이드 없이 실제 출력하여 본인이 직접 글씨를 쓸 수 있는 빈 격자를 출력합니다.  
   * **정밀 조정:** 글씨의 출력 위치 오차를 슬라이더로 조절할 수 있는 상하 미세조정(Y-Offset) 및 선의 불투명도 조절 기능을 지원합니다.  
   * **메타데이터 자동 보존:** 사용자가 입력한 제목, 원문 및 원고지색 등의 설정 값 그리고 마우스로 용지 위의 학교명, 학년/반, 이름, 꼬리말 영역을 클릭해 직접 수정한 정보는 localStorage에 즉시 안전하게 백업됩니다.

## **🛠️ 성능 및 아키텍처 혁신 (Performance & Architecture)**

전체 재생성 방식(replaceChildren)에서 탈출하여 **"고장 난 창문만 교체하는" 증분 렌더링 시스템**을 직접 구현했습니다.

### **1\. 전역 상태 캡슐화 (AppState)**

모든 전역 상태(텍스트, 메타데이터, 색상, 방향, 줌 상태 등)를 독립된 파편 변수가 아닌 단일 소스(Single Source of Truth)로 결합하여 관리합니다.

### **2\. 가상 뼈대 렌더 및 메모리 엘리먼트 풀 (DOM Element Pooling)**

renderPages() 내부에서 페이지 총량의 증감을 즉각 계산한 뒤, 부족한 페이지만 생성하고 잉여 페이지는 DOM 트리에서 순차적으로 제거합니다. 이미 존재하며 구조가 변하지 않은 페이지의 DOM 노드는 그대로 유지한 채 데이터 매핑 정보만 덮어씌웁니다.

## **🛡️ 시큐어 코딩 및 보안 (Secure Coding & XSS Defenses)**

이 앱은 .json 설정 파일 가져오기(importSettings) 및 외부 텍스트 임포트를 자유롭게 허용합니다. 만약 해커가 제목이나 이름 필드에 \<img src=x onerror=alert(1)\> 등의 악성 주입 구문을 실어 외부 저장용 .json으로 유포하더라도 완벽히 방어합니다.

* **textContent 우선주의:** 지면의 머리말, 제목, 꼬리말 정보를 바인딩할 때 위험한 innerHTML을 전면 배제하고 HTML 분석을 수행하지 않는 브라우저 내장 API textContent만을 주입합니다.  
* **XSS 방어 필터 (escapeHTML):** 부득이하게 SVG 내부 텍스트 콤보 조판을 위해 문자열 리터럴을 구성하는 부위에는 HTML 특수 문자(&, \<, \>, ", ')를 완전 격리 전치하는 탈출 필터를 거쳐 파싱하도록 이중 차단막을 형성했습니다.

## **📂 파일 구조 및 설치 (How to Run)**

* 단 하나의 **index.html** 파일로 구성되어 있어 복잡한 npm install, build 가동 단계가 전혀 필요하지 않습니다.  
* 브라우저에서 index.html 파일을 더블 클릭하여 실행하거나 정적 서버(VS Code Live Server 등)를 띄워 즉시 구동할 수 있습니다.  
* 외부 네트워크와의 소통은 구글 폰트(Noto Serif/Sans KR/JP) 로드 및 Tailwind CSS CDN 가동 외에 없으므로 완전한 단독 로컬 오프라인 실행이 가능합니다.

## **🖨️ 인쇄 설정 필수 체크 리스트**:  
  1. **배경 그래픽 인쇄**: 반드시 '켬' 또는 '체크 활성화'해야 원고지의 색상 테두리와 가운데 가이드 점선이 또렷하게 출력됩니다.  
  2. **여백**: 기본 여백이 넓어 짤릴 경우 **'없음'** 또는 '최소 여백'으로 설정해 주세요.  
  3. **머리글과 바닥글**: 브라우저 인쇄 창에서 이 옵션을 '해제(체크 해제)'해 주셔야 인쇄 일시, 웹 주소 같은 불필요한 정보가 출력물 상하단에 묻지 않습니다.
